Protocol Selection

Choose a communication protocol to test your hardware

I2C

Inter-Integrated Circuit communication protocol for short-distance communication with sensors and displays.

SPI

Serial Peripheral Interface for high-speed synchronous communication with displays and memory devices.

UART

Universal Asynchronous Receiver-Transmitter for serial communication with sensors and modules.

PWM

Pulse Width Modulation for controlling servos, LED brightness, and motor speeds.

GPIO

General Purpose Input/Output for controlling LEDs, buttons, sensors, and basic digital devices.

ADC

Analog to Digital Converter for reading analog sensors like potentiometers and light sensors.

RS485

Differential serial communication for long-distance, noise-resistant data transmission.
